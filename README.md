# XBookmark
Bookmark Plugin for Xcode

# Feature
- Toggle Bookmark
- Show Bookmarks
- Next Bookmark
- Previous Bookmark

# How To Install

- Step 1 : Install [Alcatraz](http://alcatraz.io).
- Step 2 : Search `XBookmark` from Window->Package Manager and click `Install`.
- Step 3 : Restart Xcode.

Now, you can see menus in the `Edit` Menu.
